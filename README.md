# Portfolio Optimization Using CAPM & Modern Portfolio Theory

This project applies CAPM and Modern Portfolio Theory to optimize a portfolio of selected financial stocks.

## Key Features
- Expected return estimation using CAPM  
- Efficient frontier construction  
- Maximum Sharpe ratio and minimum volatility portfolios  
- Risk analysis using VaR and CVaR  
- Skewness and kurtosis analysis
